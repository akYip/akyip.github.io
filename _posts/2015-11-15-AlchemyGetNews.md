Alchem API offers Get News API which enables you to get filtered news based on positive/negative score.

Hre's sample code to get news which has more than 0.5 score.
 

~~~~
$APPKEY="YOU APP KEY";
$KEYWORD="China";//put your keyword here


$url = "https://access.alchemyapi.com/calls/data/GetNews?outputMode=json&amp;start=now-1d&amp;end=now&amp;count=5&amp;".
"q.enriched.url.title=".$KEYWORD."&amp;q.enriched.url.enrichedTitle.docSentiment=|type=positive,score=&gt;0.5|&amp;return=enriched.url.url,enriched.url.title&amp;apikey=".
$APPKEY;


$curl = curl_init();

curl_setopt($curl, CURLOPT_POST, true);
curl_setopt($curl, CURLOPT_URL, $url);
curl_setopt($curl, CURLOPT_RETURNTRANSFER, true);
curl_setopt($curl, CURLOPT_SAFE_UPLOAD, true);
curl_setopt($curl, CURLOPT_SSL_VERIFYPEER, false);

$result = curl_exec($curl);

curl_close($curl);
print $result;
~~~~
